# Resonance-Kit MCP Apps

Interactive UI for AI-readiness codebase auditing.

## UI Resources

| URI | Description |
|-----|-------------|
| `ui://resonance-kit/dashboard` | Resonance scores, context cost, cohesion, DMS compliance |

## Usage

```javascript
import { registerResonanceAppsResources } from './mcp-apps/resonance-dashboard.js';
registerResonanceAppsResources(server);
```

## Preview

```bash
npx serve mcp-apps/
```
